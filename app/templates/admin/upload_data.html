Please ensure your Excel file (`.xlsx`) is formatted correctly. The first row must be a header with the following columns:

staff_id, patient_id, first_name, last_name, department, gender, date_of_birth, contact_phone, email_address, race, nationality

  • Rows with missing `staff_id` or `first_name` will be skipped.
